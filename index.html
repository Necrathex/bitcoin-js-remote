bitcoin-js-remote by tcatm

A JavaScript Remote for Bitcoin

bitcoin-js-remote is a user interface for Bitcoin written in JavaScript. It does not require any server side dependencies other than bitcoind.

It works in most modern browsers as well as on Android and iPhones.

You can try out this demo or even use it to connect to your own bitcoind:


Start / Demo

Patching bitcoind

If you want bitcoin-js-remote with your own bitcoind you need to apply a small patch (this will soon be included in the official binary).

CORS patch

Features

  • Android and iPhone support
  • List transactions
  • Send and receive Bitcoins
  • Supports multiple accounts

Missing Features

  • no SSL support (please don't use it over the internet yet!)
